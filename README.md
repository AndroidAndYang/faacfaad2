# faacfaad2

Build faac and faad2 on MacOS.

* faac: Include libfaac and header's file etc.

* faad: Include libfaad and header's file etc.

* fdk-aac: Include lib and header's file etc.

* MZAudioCodec: Simple Code for iOS.
